    之前用传统的 ，script引入js插件：
        1- 同步加载js文件
        2- 文件之间会依赖关系，当文件过多时，难以维护

模块化：
     1- 加载文件使用异步的方式
     2- require.js 会帮助我们管理模块之间依赖关系

     定义模块：
        define(模块名称, [依赖项], function () {
            //模块代码

            //导出项
            return  导出项；
        });
     引入模块：
        require([引入模块路径(不加.js)], function () {
            //当指定模块加载完成后回调函数
        });
